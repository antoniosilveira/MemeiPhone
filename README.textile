h1. MemeiPhone

h2. About

*MemeiPhone* is an iPhone app for "Yahoo! Meme":http://me.me, a light blogging platform from Yahoo! that is available in several languages including English, Portuguese, Spanish, Bahasa Indonesia and Traditional Chinese.

h2. Features

* Yahoo! oAuth 1.1 Authentication
* Create new posts on your Meme using Flashlight suggested content (photos, videos, web pages and tweets)
* Create new posts with pictures taken with your iPhone (or pictures from your phone gallery)

h2. Requirements

* iOS 4.2 SDK
* Titanium Developer SDK 1.2.2
* Titanium Mobile SDK 1.6.1

h2. How to

h3. Start the application

<pre>$ make run</pre>

h3. Run tests

<pre>$ make test</pre>

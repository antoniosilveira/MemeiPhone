This project is an iPhone app for Yahoo! Meme, a light blogging platform from Yahoo! that is currently available in several languages including English, Portuguese, Spanish, Bahasa Indonesia and Traditional Chinese.

This app has the objective to offer a native looking way to navigate and consume content from user's dashboard on Meme.

Development powered by Titanium Mobile framework that allows development for iOS platform using Javascript instead of ObjC.

h2. Requirements

- iOS 4.2 SDK
- Titanium Developer SDK 1.2.1
- Titanium Mobile SDK 1.6

h2. Features

- oAuth 1.1 Authentication
- Navigate on Meme's dashboard
- Create new posts with Flashlight suggested content and live preview of links
- Repost any Post from within the App
